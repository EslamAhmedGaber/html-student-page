Twoâ€™s Complement Notation and Arithmetic

Explanation: Two's complement notation is the most common method for representing signed integers in binary. It allows both positive and negative numbers to be represented using the same binary format.
  • The most significant bit (MSB) is the sign bit: 0 for positive, 1 for negative.
  • To represent a negative number, take the binary representation of the positive number, invert all bits, and add 1 to the result.

Example: Converting -5 to Two's Complement (4 bits)

  • Step 1: Write the positive version of the number (5) in binary: 0101.
  • Step 2: Invert the bits: 1010.
  • Step 3: Add 1 to the result: 1011.
  • Result: The two's complement of -5 is 1011.
Important Tip: Always check the sign of the result when performing binary addition, especially with fixed-length binary numbers, to detect overflow.

C++ Example: Simulating Two's Complement Conversion


#include <iostream>
using namespace std;

int twoComplement(int num, int bits) {
    int mask = (1 << bits) - 1;
    return (~num + 1) & mask;
}

int main() {
    int num = 5;  // Positive number
    int bits = 4; // 4-bit representation
    int twoComp = twoComplement(num, bits);
    cout << "Two's complement of " << num << " (4-bit) = " << twoComp << endl; // Output: 11
    return 0;
}
